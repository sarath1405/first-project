Array :

The array is a collection of homogeneous objects and this array container is defined for constant size arrays or (static size). This container wraps around fixed-size arrays and the information of its size are not lost when declared to a pointer.

This C++ STL array is a kind of sequential container and is not used extremely in regular programming or in competitive programming but sometimes its member function provides an upper edge to it over the regular normal array that we use in our daily life. So, we are discussing some of the important member function that is used with such kind of array:

Declaration:

array < 'DataType', 'size of array' > 'name of the array';

example: array < int, 5 > arr1;

here are the list of functions associated with array
  1. begin( ) and end( ) function: Both these begin( ) and end( ) functions returns the iterator to the beginning and ending of the array respecively.

  2. [ ] Operator: This is similar to the normal array, we use it to access the element stored at the particular index.

  3. front( ) and back( ) function: These methods are used to access the first and the last element of the array directly.

  4. swap( ) function: This swap function is used to swap the content of the two arrays.

  5. empty( ) function: This function is used to check whether the declared STL array is empty or not, if it is empty then it returns true else false.

  6. size( ) or max_size( ) and sizeof( ) function: Both size( ) or max_size( ) are used to get the maximum number of indexes in the array while sizeof( ) is used to get the total size of array in bytes.

  7. data( ): This function returns the pointer to the first element of the array object. Because elements in the array are stored in contiguous memory locations. This data( ) function return us the base address of the string/char type object.
