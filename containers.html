Containers :

A container is a holder object that stores a collection of other objects (its elements). They are implemented as class templates. The container manages the storage space for its elements and provides member functions to access them, either directly or through iterators. There are 2 types of containers, Sequential containers and Associative containers. Listed below are the different types of containers in cpp.
